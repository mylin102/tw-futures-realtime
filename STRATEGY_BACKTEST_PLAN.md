# 📊 策略優化回測計畫 (Backtest Optimization Plan)

此計畫旨在透過「參數矩陣掃描」找出台指期 (TMF) 最穩健的獲利組合。

## 1. 實驗維度 (Test Matrix)

我們將針對以下四個關鍵維度進行交叉測試：

| 維度 | 參數名稱 | 測試值 (Values) | 目的 |
| :--- | :--- | :--- | :--- |
| **靈敏度** | `squeeze_length` | 10, 14, 20 | 測試不同波段長度的能量擠壓效果 |
| **進場門檻** | `entry_score` | 60, 70, 80 | 尋找過濾假訊號與捕捉趨勢的平衡點 |
| **環境過濾** | `regime_filter` | none, mid, macro | 驗證不同週期均線對趨勢判斷的準確度 |
| **風險空間** | `stop_loss_pts` | 30, 40, 50 | 找出最能容忍震盪且不重傷本金的停損點 |

## 2. 比較邏輯 (Comparison Logic)

報告將依據以下權重進行綜合評分：
1.  **總獲利 (Net Profit)**: 首要指標，需為正值。
2.  **勝率 (Win Rate)**: 評估策略的心理壓力與執行難度。
3.  **最大回撤 (MDD)**: 評估策略的風險控制能力。
4.  **獲利因子 (Profit Factor)**: 總獲利除以總虧損，需 > 1.2。
5.  **交易頻率**: 排除過度擬合 (Overfitting) 的極端案例。

## 3. 輸出報告格式

最終報告將包含：
- **Executive Summary**: 推薦的最佳參數組合。
- **Sensitivity Analysis**: 改變單一變數對損益的影響趨勢圖。
- **Top 5 Ranking Table**: 績效排名前五名的組合明細。
- **Equity Curve PK**: 前三名策略的資產走勢對照圖。

## 4. 預期目標
確認在加入 Pullback 邏輯後，是否能透過「中線過濾 + 高 Score」維持穩定的獲利期望值。
